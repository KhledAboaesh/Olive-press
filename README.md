# Olive Press Manager

نظام إدارة معصرة زيتون محلي بواجهة عربية حديثة.

## الميزات
- إدارة الزبائن، الاستلام، الفواتير، الديون، التعديلات
- تخزين البيانات في ملفات JSON
- طباعة الفواتير من قالب HTML
- نسخ احتياطي إلى ZIP
- دعم ثيم فاتح/داكن
- زر إعادة ضبط البيانات

## التشغيل
```powershell
pip install -r requirements.txt
python app.py
```
